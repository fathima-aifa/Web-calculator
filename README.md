# Web Calculator

A simple calculator built using HTML, CSS, and JavaScript.

## About

This repository was created while learning frontend development and experimenting with AI-assisted coding tools (Gemini CLI).

The calculator app and project setup were generated using AI, including parts of the project structure and GitHub integration.

It was mainly used to understand how a basic web application is structured and how AI can assist in development workflows.

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Simple user interface
- Runs entirely in the browser

## Tech Stack

- HTML
- CSS
- JavaScript
